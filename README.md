# 🚀 WordVault – Interactive English Learning Tool

WordVault is a modern, interactive web-based English learning tool designed to help users improve their vocabulary with ease. It provides categorized words, Hindi meanings, and real-time progress tracking.

---

## ✨ Features

* 📚 200+ essential English words
* 🧠 Category-wise learning (Pronouns, Verbs, Adjectives, etc.)
* 🔍 Smart search (English + Hindi support)
* ✅ Click-to-mark words as "learned"
* 📊 Progress tracking system
* 💾 LocalStorage support (data saved automatically)
* 🎨 Beautiful modern UI with animations
* 📱 Fully responsive design

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Advanced UI + Animations)
* JavaScript (DOM + LocalStorage + Logic)

## 🧠 How It Works

* Click on any word → it gets marked as **learned ✅**
* Progress bar updates automatically
* Data is saved in your browser (no login required)
* Use search to quickly find words

## 🎯 Purpose of the Project

This project solves a common problem faced by learners:

> Difficulty in remembering and practicing commonly used English words.

WordVault provides a simple, interactive, and engaging way to learn vocabulary effectively.
